# Reforzamiento de TypeScript y React

Proyecto educativo con ejercicios de TypeScript y conceptos fundamentales de React.

## 📋 Requisitos Previos

- Node.js (versión 16 o superior)
- npm o yarn

## 🚀 Instalación y Ejecución

### 1. Instalar dependencias

```bash
npm install
```

O si usas yarn:

```bash
yarn install
```

### 2. Ejecutar en modo desarrollo

```bash
npm run dev
```

O con yarn:

```bash
yarn dev
```

El proyecto se abrirá automáticamente en `http://localhost:5173` (o en el puerto que Vite asigne).

### 3. Compilar para producción

```bash
npm run build
```

O con yarn:

```bash
yarn build
```

## 📁 Estructura del Proyecto

```
src/
```
